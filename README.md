---

## 📸 Preview

```
_ _ _ _ _
Guess a letter: a
Correct! 🎉
```

---

## 📌 About

This game randomly selects a word and challenges the player to guess it letter by letter within **6 attempts**.

It helps beginners understand:

* Loops
* Conditions
* Strings
* Lists
* Random module

---

## 🚀 How to Run

```bash
python hangman.py
```

---

## 🎯 How to Play

* A hidden word appears as blanks `_ _ _ _ _`
* Enter one letter at a time
* Correct guess reveals the letter ✅
* Wrong guess reduces chances ❌
* You have **6 lives**

---

## 🛠 Requirements

* Python 3.x
* No external libraries required
